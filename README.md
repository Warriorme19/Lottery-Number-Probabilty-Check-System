# Lottery Probability Simulator

- Simulates a six-digit lottery system using C++.
- Generates a random lottery number.
- Accepts a six-digit number from the user.
- Compares the user's number with the generated number.
- Displays whether the user wins or loses.
- Calculates the theoretical probability of winning.
- Demonstrates random number generation using `rand()`.
- Uses basic C++ concepts like input/output and conditional statements.
- Built to practice programming logic and probability concepts.
